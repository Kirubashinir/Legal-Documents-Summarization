Legal documents are inherently complex, lengthy, and filled with domainspecific terminology, making them difficult for non-experts to interpret and analyze efficiently. This project presents an AI-powered Legal Document Summarization System designed to simplify and enhance understanding using advanced Natural Language Processing (NLP) techniques and BERT-based deep learning models. The system automatically extracts key information and generates concise, context-aware summaries while preserving the original legal meaning and intent.The proposed system follows a structured pipeline including document input processing, text preprocessing, semantic analysis, and intelligent summarization. It leverages tokenization, Named Entity Recognition (NER), and transformer-based contextual learning to identify critical entities such as legal parties, dates, clauses, and obligations. Using an abstractive approach, it generates meaningful summaries rather than simply selecting sentences, ensuring improved accuracy, coherence, and contextual relevance.To enhance accessibility, the system integrates multilingual capabilities using Neural Machine Translation (NMT), allowing users from diverse linguistic backgrounds to understand legal content effectively. An interactive interface enables users to upload documents, explore summaries, and obtain context-aware explanations through query-based interactions, transforming it into an intelligent legal assistant.The system also emphasizes data security by processing sensitive information in a controlled environment without permanent storage. By automating summarization, it reduces time, effort, and dependency on legal experts for document analysis# Legal-Documents-Summarization
